
# **API Disney**
 
 API Development to explore the Disney world, this allows to know and modify the characters that compose it and which movies they participated in.

### **Technologies used:**

    - sendgrid
    - bcryptjs
    - express
    - jsonwebtoken
    - MySQL
    - sequelize
